# BTL_ATTT_223
Thời gian: 4 tuần
Thành viên: Tân - Hiên - Thuận - Quý
Ngôn ngữ: SQL Oracle

Tuần 1:
Vote xong thì nhóm viết mô tả về ứng dụng cần thực hiện:
- Ứng dụng dùng làm gì, các nghiệp vụ cần thực hiện là nghiệp vụ nào
- Gồm những người dùng nào sử dụng
- Thông tin cần lưu là gì
- Liệt kê các đối tượng, các quan hệ giữa các đối tượng
- Vẽ sơ biểu đồ quan hệ giữa các đối tượng (Logical)
- Chính sách bảo mật áp dụng
  => tương ứng với các người dùng họ được phép truy cập thông tin nào
  => MAC, DAC

Tuần 1:
Viết database:
- Từ mô hình Logical vẽ mô hình Relation sau đó viết code SQL tạo bảng, ràng buộc
  => Làm tay hoặc dùng SQL developer, SQL Modeler để vẽ và tự tạo code trên Oracle
- Viết các trigger, function, procedure nếu cần
- Nhập dữ liệu vào các bảng để test

Tuần 2 - Tuần 3:
Thực hiện bài lab:
- Dùng database trên thực hiện các bài Lab:
  => Lab 4 - 5 : RLS (Row level Security
  => Lab 6 - 7 - 8 : Oracle Label Security
  => Lab 9 : Standard Auditing
  => Lab 10 : Fine-Grained Auditing

Tuần 2 - Tuần 3
Viết bảo mật cho ứng dụng:
- Bổ sung dabase
- Dùng các kiến thức lúc làm lab viết bảo mật ứng dụng
- Viết các câu lệnh để test loại bảo mật này

Tuần 4:
Viết giao diện đơn giản cho ứng dụng;
- Kết nối database
- cho thấy được các chính sách bảo mật

Tuần 4:
Viết sơ báo cao:
